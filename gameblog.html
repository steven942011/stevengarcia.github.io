Aplicacion Web para un Blog Gaming

BlogGame una aplicación web que cuenta con diferentes módulos atractivos para un blog en el cual los usuarios registrándose podrán compartir sus experiencias, datos o gustos por los juegos.
Los diferentes módulos con los que cuenta son:

  • Registro de Usuarios
  • Ingreso de Usuario por correo
  • Creación de categorías
  • Crear artículos por categoría
  • Visualización por categoría
  • Control de acceso a categorías por usuario
  • Actualización y eliminación de artículos por usuario único
  • Actualización de datos de usuario
  • Registro en base de datos
  • Búsqueda de artículos

    • Tecnologias:

  • HTML
  • JAVASCRIPT
  • CSS3
  • PHP
  • MYSQL
